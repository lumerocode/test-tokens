# 🔨 Auction Design System

Este sistema fue construido bajo la metodología de **Atomic Design**, priorizando la escalabilidad mediante **Design Tokens**.

### 🛠️ Stack
- **React + Vite**: Para un entorno de desarrollo ultra rápido.
- **Tailwind CSS v4**: Gestión de estilos mediante variables nativas.
- **Storybook**: Documentación interactiva en la nube.

### 🎨 Design Tokens
- `auction-primary`: Color de marca para acciones principales.
- `auction-urgent`: Estado de alerta para subastas finalizadas en < 5min.

### ☁️ Cloud & Producción
El sistema está desplegado automáticamente en **Vercel** mediante un pipeline de CI/CD conectado a este repositorio.
